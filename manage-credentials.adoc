= 移除認證資料
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
如果您不再需要一組認證、可以從 Workload Factory 中刪除這些認證。您只能刪除與 ONTAP 檔案系統的 FSX 無關的認證。

.步驟
. 在 Workload Factory 主控台中、選取 * 帳戶 * 圖示、然後選取 * 認證 * 。
. 在 * 認證 * 頁面上、選取一組認證的動作功能表、然後選取 * 移除 * 。
. 選取 * 移除 * 以確認。
